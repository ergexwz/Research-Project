# 数据说明

原始数据为保密数据，未包含在仓库中。请联系项目方获取。

## 数据格式

- **文件格式**: CSV（GBK 编码）
- **结构**: 71 列，包含输入特征和层次化输出标签
- **放置路径**: 将 CSV 文件放入 `initdata/` 目录，并在 `configs/hecto_config.yaml` 中配置 `data.raw_path`

## 输入列（22 数值 + 24 类别）

### 数值特征（22列，含6组三元组 max/nor/min）
- `计算CV（最大）`、`计算CV（正常）`、`计算CV（最小）`
- `入口温度（最大）`、`入口温度（正常）`、`入口温度（最小）`
- `流量（最大）`、`流量（正常）`、`流量（最小）`
- `阀前压力（最大）`、`阀前压力（正常）`、`阀前压力（最小）`
- `阀后压力（最大）`、`阀后压力（正常）`、`阀后压力（最小）`
- `压差（最大）`、`压差（正常）`、`压差（最小）`
- `关闭压力`、`设计温度`、`上游管道口径`、`供气压力`

### 类别特征 — 低基数（21列）
`流体状态`、`阀门型式`、`阀体型式`、`流量特性`、`泄露等级`、`压力等级`、`硬化保护`、`法兰型式`、`连接方式`、`驱动方式`、`环境温度`、`防护等级`、`阀体材质`、`阀芯材质`、`阀座材质`、`阀杆材质`、`填料`、`温度单位`、`流量单位`、`压力单位`、`制造品牌`

### 类别特征 — 高基数（3列）
`位号`、`流体名称`、`管道材质`

## 输出列（21个，3级层次化）

### Level 1: 产品系列
`产品系列`（14类）

### Level 2: 产品型号
`产品型号`（64类，受系列约束）

### Level 3: 分类规格（17列）
`公称通径`、`压力等级.1`、`法兰型式.1`、`泄露等级.1`、`流量特性.1`、`阀盖型式`、`阀体材质.1`、`阀芯材质.1`、`阀板材质`、`阀座材质.1`、`阀盖材质/轴材质`、`密封环材质`、`填料.1`、`填料型式`、`流向`、`波纹管材质`、`波纹管型式`

### Level 3: 回归规格（3列）
`额定CV/KV`、`阀座直径`、`额定行程`

## 预处理说明

`data/preprocessing.py` 中的 `DataPreprocessor` 会自动执行：
1. GBK 编码加载
2. 数值清洗（inf/NaN 处理、物理范围裁剪）
3. 压力单位统一（kPa/bar/psi 等 → MPa）
4. log1p 压缩 + StandardScaler 标准化
5. LabelEncoder 编码所有类别特征和输出标签
